## Install poetry

```
curl -sSL https://install.python-poetry.org | python3 -
```

## Get into the shell

```bash
poetry shell
```

## Get virtual env info

```bash
poetry env info
```

## add a package

```bash
poetry add <package_name>
```

## Install dependencies

```bash
poetry install
```
